Why Link Checking Is Important

  • In an ideal world, URLs would never change (see Cool URIs don't change and URL as UI)
  • Unfortunately, links do break
  • In Q4 2017, 12,990+ broken links found within 562,361 pages on UK government web sites.
  • Broken links make your site look unprofessional, reduce user confidence and can lead to lost revenue
  • Broken internal links are even worse
  • Therefore all sites should be link checked before being made public and regularly after go live

How To Link Check

  • Link check up to 150 pages using W3C Link Checker
  • Link checking larger sites needs locally installed software
  • Xenu is recommended because it is free and simple
  • Xenu can also link check sites that are only on your computer (good for testing)
  • BLT and Integrity are similar and available for Macs

Common Error Codes (HTTP Status Codes)

  • 404 - Not Found
    Permanent. Link is broken.
  • 401 - Unauthorized
    Access requires a username and password.
  • 403 - Forbidden
    Access is not allowed (e.g. because access is restricted to specific machines or users).
  • 503 - Service Unavailable
    Perhaps temporary. The Web site may be undergoing maintenance or there may be a network problem.
  • 500 - Internal Server Error
    Usually temporary. The Web server has found an error whilst trying to display the page.
